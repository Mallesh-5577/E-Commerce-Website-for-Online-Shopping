# 🛒 E-Commerce Website

## 📌 Project Overview
The **E-Commerce Website** is a simple web application designed to simulate an online shopping platform.  
It allows users to browse products, view detailed product information, add items to a shopping cart, and proceed to checkout.

This project demonstrates the use of **HTML, CSS, and JavaScript** to build an interactive and user-friendly shopping experience.

---

## 🎯 Project Objectives
- To develop a basic online shopping website.
- To provide an interface where users can explore products easily.
- To implement a shopping cart system using JavaScript.
- To understand front-end web development concepts.

---

## 🛠️ Technologies Used

- **HTML5** – Website structure
- **CSS3** – Styling and layout
- **JavaScript** – Dynamic functionality
- **Visual Studio Code** – Development environment
